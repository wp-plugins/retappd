This plugin displays a user's Untappd checkins in a post or a page by the use of a shortcode.

== Description ==

This plugin can be used to display a user's most recent checkins in a post or a page by the use of a shortcode. You can choose what pieces of Untappd checkin information that you would like to display in plugin's settings. Example usage of the shortcode is as follows: [retappd].

== Installation ==

1. Upload the full directory into your wp-content/plugins directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Visit your Retappd Settings (Settings - Retappd)

4. Configure required Retappd options

5. Place the Retappd shortcode on the pages that you want to give users access to the Untappd user feed: [retappd]
